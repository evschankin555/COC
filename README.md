## Автокликер для игры

Автоматический кликер для игры, который находит и кликает по кнопке "Надеть" в правой части экрана, не мешая работе пользователя.

### Особенности

- Автоматически определяет правую половину экрана для поиска кнопки
- Работает незаметно - возвращает курсор на исходную позицию после клика
- Не мешает работе - ищет кнопку только в правой части экрана (где игра)
- Использует существующее изображение кнопки `btn.png`

### Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

### Использование

1. Убедитесь, что файл `btn.png` находится в папке со скриптом
2. Запустите скрипт:
```bash
python autoclicker.py
```

3. Остановка: нажмите `Ctrl+C`

### Настройки

В файле `autoclicker.py` можно изменить следующие параметры:

- `CHECK_INTERVAL = 0.1` - интервал проверки в секундах (10 раз в секунду)
- `CONFIDENCE = 0.8` - точность поиска (0.8 = 80%, можно снизить до 0.7 если не находит)
- `CLICK_DELAY = 0.01` - задержка после клика (минимальная для мгновенного возврата)

### Как это работает

1. Скрипт автоматически определяет размер экрана
2. Ограничивает поиск кнопки правой половиной экрана
3. При нахождении кнопки выполняет мгновенный клик
4. Возвращает курсор точно на исходную позицию
5. Продолжает работу незаметно для пользователя

### Советы

- Если кнопка не находится, снизьте `CONFIDENCE` до 0.7
- Для ускорения можно включить `grayscale=True` в функции `locateOnScreen`
- Убедитесь, что игра видна в правой части экрана
